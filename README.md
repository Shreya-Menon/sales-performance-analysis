# Sales Performance & Customer Behavior Analysis

## Objective
Analyze sales data to identify trends, profitability drivers, and actionable insights.

## Dataset
Sample Superstore dataset (public dataset)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Technology category leads in revenue
- High discounts reduce profit
- Central region needs attention

## Business Recommendations
- Optimize discount strategy
- Focus on high-margin products
- Improve regional performance

SQL analysis performed using SQLite-
• Technology category contributes the highest revenue
• Central region shows poor profitability
• High discounts significantly reduce profit
• Tables and Bookcases are consistent loss-makers
• Standard Class shipping is most cost-efficient
